# AfriTalent

## Description

AfriTalent est une plateforme vitrine fictive mettant en relation des freelances tech africains et des entreprises à la recherche de talents qualifiés.

Ce projet a été réalisé dans le cadre d'un projet académique individuel afin de mettre en pratique les technologies fondamentales du développement web front-end.

---

## Objectifs du projet

* Créer un site vitrine moderne et responsive
* Présenter des profils de freelances tech africains
* Mettre en avant les services et plans tarifaires
* Fournir une expérience utilisateur agréable grâce aux animations et interactions JavaScript

---

## Technologies utilisées

* HTML5
* CSS3
* Bootstrap 5
* JavaScript Vanilla

---

## Fonctionnalités

### Interface utilisateur

* Navbar responsive
* Hero section moderne
* Cartes freelances
* Présentation des catégories
* Témoignages clients
* Plans tarifaires
* FAQ avec accordéon Bootstrap
* Formulaire de contact

### Expérience utilisateur

* Dark Mode avec sauvegarde LocalStorage
* Navbar dynamique au scroll
* Animations Fade-In au défilement
* Compteurs statistiques animés
* Filtrage dynamique des freelances
* Validation JavaScript du formulaire
